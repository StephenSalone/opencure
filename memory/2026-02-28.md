### The Project: OpenCure (opencure.life)
Open-source AI pipeline for drug repurposing targeting neglected tropical diseases.

### Tech Stack Built
- conda env `drugdiscovery`: OpenMM 8.4 + OpenFF 2.0 + RDKit + PDBFixer + OpenBabel
- AutoDock Vina 1.2.7 (Apple Silicon binary) in tools/
- Semantic Scholar API wrapper (tools/semantic_scholar.py)
- repurposing_agent.py — automated screening pipeline
- Full MD simulations: HIV protease (44,734 atoms), Aspirin, Niclosamide vs Cruzain (284,797 atoms)
