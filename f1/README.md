# OpenCure F1 Performance Lab

Using the same computational pipeline that found longevity drug candidates,
applied to Formula 1 performance materials.

## Targets
1. Tire Compound Optimization — polymer MD for grip/heat/wear
2. Fuel Additive Screening — combustion efficiency molecules
3. Aerodynamic Surface Coatings — drag reduction materials

## Pipeline
- RDKit: molecular preparation
- OpenMM: molecular dynamics simulation
- AutoDock Vina: binding/interaction screening
- Semantic Scholar: published F1 materials literature

## Novel Aspect
Never before analyzed with Nova (Anthropic AI) + Grok (xAI) collaboration.
All results open source. github.com/StephenSalone/opencure
